Publishing Plugins

Everything you need to know before publishing a plugin on the QGIS Plugin Repository.

Getting started

To publish a plugin you need an OSGeo ID. Once you have one, go to the Upload a plugin page and follow the upload wizard. Your plugin will be queued for review — see the Approval Process page for what to expect.
